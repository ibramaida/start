Layout utilities

wrapper

Creative wrapper that use a grid hack to be able to choose the size of the content. the default size is 900px can adjusted with "--content-max-width" custom property.

breakout

Break the content a little bit out, this size can be adjusted with those custom properties "--breakout-max-width" and "--breakout-size".

full-width

Make full width sections while maintaining the content within the max content size, can be using for photos as wel.

flex

provide display flex with default gap "1rem" can adjust with "--gap" custom property and "align-items: center".

grid

provide display grid with default gap "1rem" with default gap "1rem" can adjust with "--gap" custom property.

equal-columns

make equal columns using display grid with default gap "1rem" with default gap "1rem" can adjust with "--gap" custom property, the columns are minimum "200px" wide and maximum "1fr".

d-block

provide "display: block".

flow

give all direct children default "margin-top: 1rem" behalf the first one, the margin can adjust with "--flow-space" custom property.

flow--space-small

give all direct children default "margin-top: 0.75rem" behalf the first one, the margin can adjust with "--flow-space" custom property.

section-space

it use to add enough space between sections which give all direct children default "margin-top: 3rem" behalf the first one, the margin can adjust with "--section-space" custom property.

Accessability classes

sr-only

Make the element visually hidden only for the screen reader.

skip-to-content

Make skip to content button that trigger when a tab pressed on keyboard.
